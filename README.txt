PRIEDS TMS MODULAR HTML

Files:
- index.html: menu utama
- master.html
- operational.html
- report-overview.html
- integration.html
- risk-control.html

Recommended method:
1. Pastikan semua file tetap berada di folder yang sama.
2. Jalankan: python start-server.py
3. Browser akan membuka http://localhost:8000/index.html

Menjalankan melalui local server memastikan semua module menggunakan origin dan localStorage yang sama, sehingga data Master, Operational, Report, Integration, dan Risk & Control tetap sinkron.
